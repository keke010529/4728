## 🎯 仓库简介

本仓库致力于收集、整理和分享 **GB-T 4728《电气简图用图形符号》** 系列国家标准的各个版本及相关资源，为电气工程、自动化、电子设计等领域的工程师、学生和研究人员提供便捷的标准查询和使用体验。

> ⚠️ **重要声明**：本仓库所有标准文件均来源于公开渠道，仅供学习、研究和技术交流使用。如需商业使用，请购买正式出版物或联系中国标准出版社获取授权。
